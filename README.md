# 🐑 crontab-admin

- Linux Cronatb 서비스를 관리할 수 있는 프로그램입니다.
- local -> single remote -> multi remote 순으로 개발 예정입니다.
- CI/CD 연동을 통해 관리 프로그램 개발시 자동 테스트 및 배포 적용 예정입니다.

# 📚 기술 스택

- CentOS 7.9 / Oracle DB 12C / C++
- cri-o / kubernetes v1.25
- github workflow / ArgoCD

# 🌟 주요 기능
### [Application]
